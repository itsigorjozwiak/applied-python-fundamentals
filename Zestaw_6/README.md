# Applied Python Fundamentals
## Task set 6

> The sixth set focuses on 2D computer graphics, physics simulations, game state management, and AI-assisted code generation using `turtle` and `pygame`.

* **Task 1:** Interactive parametric curve visualization (Lissajous figures) with real-time modulation of frequencies and phase shifts, using the standard graphics library `turtle`.
* **Task 2:** 2D physics simulation of a bouncing ball within a gravitational field, featuring perfectly elastic screen-edge collisions and real-time kinematic controls via `pygame`.
* **Task 3:** Refactoring a Ping-Pong-style project into a single-player arcade game using `pygame.sprite`, implementing bounding-box collision detection, real-time keyboard input handling, scoring logic, game state transitions, and persistent high-score storage via File I/O.
* **Task 4:** Using an LLM and prompt engineering to generate a functional Paint-style drawing application in `pygame` based on a reference image.

## Setup

**Dependencies:** Python 3.x, `pygame`. *(Note: `turtle` and `math` are included in the standard Python library).*

The project uses `pygame` together with Python’s standard library, so installing pygame is enough to run everything.

```bash
# Install pygame
pip install pygame

# Example of running Task 2
cd ZADANIE2
python zadanie2.py
```
