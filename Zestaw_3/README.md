# Applied Python Fundamentals
## Task set 3

> The third set focuses on high-precision algorithm benchmarking, financial time-series analysis and interactive geospatial mapping.

* **Task 1:** High-performance Fibonacci sequence computation and benchmarking comparing iterative, matrix exponentiation and `gmpy2` approaches.
* **Task 2:** Interactive geospatial data visualization and map generation using `folium`.
* **Task 3:** Financial data acquisition and time-series analysis (e.g., BTC-USD moving averages) leveraging `yfinance`, `pandas` and Jupyter Notebooks.

## Setup

**Dependencies:** Python 3.x, `gmpy2`, `yfinance`, `pandas`, `folium`, `numpy`, `pytest`

```bash
# Install required packages
pip install -r requirements.txt

# Example of running Task 3
python ZADANIE3/zadanie3.py
```

## Tests   
Unit tests are provided for tasks containing core algorithmic logic and computational benchmarking.

```bash
# Example of running tests for Task 1
pytest ZADANIE1/tests/
```
